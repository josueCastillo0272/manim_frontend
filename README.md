# manim_frontend
This project provides a **GUI** for creating and managing animations using Manim, the mathematical animation engine. It aims to simplify the process of building Manim scenes by offering an intuitive frontend for previewing, editing, and rendering animations — without needing to work entirely from the command line.

## LaTeX-OCR
The **LatexOCR** module extracts mathematical expressions from images and converts them into LaTeX code. It combines optical character recognition (OCR) with deep learning to accurately identify and transcribe handwritten or printed math formulas, enabling easy integration into documents, editors, or animation tools like Manim.
